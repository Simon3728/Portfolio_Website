Bitcoin Sentiment Analysis

In this ongoing project, I use Tweets from X about Bitcoin to explore various methods of sentiment analysis. My goal is to understand how public sentiment influences Bitcoin's stock price. Currently, I am testing different sentiment analysis techniques.

Sentiment Analysis

For sentiment analysis, I want to compare four approaches and test their effectiveness for my project:

  1. The first one is the Twitter-roBERTa-base Sentiment Analysis Algorithm from Hugging Face. This algorithm analyzes tweets by breaking them down into smaller parts, understanding the context of each word using a technique called transformers, and then predicting whether the sentiment is positive, neutral, or negative. It’s fast and should be quite accurate, thanks to its training on 124 million tweets.
  2. Llama 3.1 8B is the latest text-based AI from Meta, and since it’s open source, I can run it locally on my computer. It also allows for a more comprehensive analysis of the tweets, such as identifying the actual emotions expressed and understanding how relevant tweets are to Bitcoin's performance.
  3. To practice implementing my own machine learning algorithms, I plan to use Support Vector Machines (SVM). SVM is a supervised learning algorithm that classifies text in sentiment analysis by finding the optimal boundary, or hyperplane, that separates different sentiment classes (such as positive and negative) in a high-dimensional space.
  4. My final approach will be to use a Random Forest algorithm. In sentiment analysis, Random Forests work by creating many decision trees, each using different random parts of the data. The model then combines the predictions from all the trees to decide if the text is positive, negative, or neutral.
